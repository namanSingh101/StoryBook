
# Story Book App

A Storybook app is a digital platform that offers a collection of interactive and illustrated stories for user, providing an engaging and educational reading experience . Users can explore a wide range of stories,to enhance the storytelling experience.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
# Run in development
npm run dev

# Run in production
npm start
```


## Environment Variables

To run this project, you will need to add the following environment variables to your config.env file

`PORT`

`MongoURI`

`GOOGLE_CLIENT_ID`

`GOOGLE_ClIENT_SECRET`


## Documentation

This app uses Node.js , Express ,MongoDB , Passport with Google OAuth for email authentication.

For making DELETE and PUT request we have used method override package

For templating I have used handlebars
